# The Brand Bar

## Descripción General

**The Brand Bar** es un sitio web interactivo que simula el funcionamiento digital de un restobar. El proyecto permite a los usuarios explorar la carta de productos, realizar pedidos mediante un carrito de compras y gestionar reservas para eventos o reuniones.

El sitio busca mejorar la experiencia del cliente al ofrecer una plataforma donde pueda conocer los productos disponibles, realizar pedidos de manera rápida y reservar espacios dentro del establecimiento sin necesidad de atención presencial.

## Funcionalidades

### Carta Digital Interactiva

* Visualización de alimentos, bebidas y cócteles.
* Organización de productos por categorías.
* Presentación visual de cada producto mediante tarjetas.

### Carrito de Compras (Funcionalidad Dinámica)

* Agregar productos al carrito.
* Visualizar el resumen de productos seleccionados.
* Actualización dinámica del pedido.
* Confirmación del pedido realizado.

### Seguimiento mediante WhatsApp

* Después de confirmar el pedido, el usuario puede continuar la comunicación mediante WhatsApp para realizar el seguimiento correspondiente.

### Book The Brand Bar

* Galería fotográfica del ambiente del restobar.
* Visualización de experiencias y opiniones de clientes.

### Registro de Usuarios (Funcionalidad Dinámica)

* Formulario para registrar nombre y correo electrónico.
* Almacenamiento temporal de los datos ingresados por el usuario.

### Sistema de Reservas (Funcionalidad Dinámica)

* Selección de fecha para la reserva.
* Ingreso del motivo o descripción del evento.
* Registro de la solicitud de reserva para futuras coordinaciones.

## Instrucciones de Uso

### Acceso al Sitio Web

El proyecto se encuentra desplegado en Netlify y puede visitarse desde el siguiente enlace:

https://brandbarcamisolano.netlify.app/

### Ejecución Local

1. Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

2. Ingresar a la carpeta del proyecto:

```bash
cd TheBrandBar
```

3. Abrir el archivo `index.html` en cualquier navegador web moderno.

No requiere instalación de dependencias adicionales.
